<<Python机器学习及时间>>读书笔记

小结：
这本书对常见的机器学习算法都做了全面的概述，包括有监督学习，无监督学习，涵盖了全面的传统机器学习算法，已经集成学习算法，没有算法进行深入探究，介绍了每种算法的使用场景，已经算法的在skit-learn的使用方法，没有涉及到数学知识，非常适合初学者
特点：
介绍基本机器学习流程，算法全面，适合初学者对整体机器学习学科以及实践有全面的了解
